The Django Movie Database Project



Motivation

The aim of the project is to build a movie database to recommend movies based on ratings of a user. There's a "My Stars" section in profile which is also your Favourite section.

All visitors are able to look at the movie list, their average and total ratings. After a user logins, the user is able to start rating movies. Then, the recommended movies are available for the user.



Backend


The project is powered by Django and, of course, Python. There are two applications in the project: movies and users. Users application is created to customize user schema with date of birth, gender and country. Movies application is created for movies schema.

SQlite is used for database.

Class Based View is mainly preferred over Function Based View in the project.

To get movie details, a third party API is used: OMDb API



Frontend


In the frontend JavaScript, JQuery, Bootstrap, CSS and HTML is used.
